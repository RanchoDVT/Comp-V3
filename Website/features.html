All the features you would want in a beginner V5 program... and then some!

Features

🌀 Gyro Calibration

Function: calibrateGyro()
Description: Easily calibrate the gyro sensor for accurate readings.
Example: calibrateGyro();
Return Value: None.

📁 Config Manager

Methods:

  • Get Current Logging Setting:
    Function: getLogToFile()
    Description: Check if logging to a file is enabled. Returns true if enabled, false otherwise.

  • Set Logging Configuration:
    Function: setLogToFile(bool logToFile)
    Parameters: logToFile (type: bool) - Set to true to enable logging, false to disable.

🔘 Button Press Detection

Function: ctrl1BttnPressed()
Description: Detect pressed buttons on the primary controller and retrieve the button pressed along with the duration.
Example: auto [button, duration] = ctrl1BttnPressed();
// OR if you only want the button!
std::string buttonPressed = std::get<0>(ctrl1BttnPressed());
Return Value: Pair of button pressed and duration.

📊 User Options Selection

Function: getUserOption()
Description: Prompt users for a setting choice and retrieve their selection.
Example Use: 

int value = getUserOption("Choose a setting", {"Option 1", "Option 2"});

Return Value: User selected option. (Ex: Option 1)

🎞️ GIF Player

Function: startGifPlayer()
Description: Initiate the GIF player for displaying animated content.
Return Value: None.

📝 Logging System

Function: loghandler()
Description: Log messages with varying severity levels (Trace, Info, Warn, Error, Fatal). Displays messages on the console and controller screen based on the log level.
Return Value: None.

🌡️ Motor Monitoring

Function: motorMonitor()
Description: Monitor motor temperature and position during competitions to ensure performance and safety.
Return Value: None.

💾 SD Card Compatibility

Requirements:

  • SDHC type recommended, formatted in FAT32.
  • Max size of 16GB.

Structure:

  • /assets: Store logo images (.png, .gif)
  • /backup: Latest source code backups
  • config.cfg: Stores robot settings
