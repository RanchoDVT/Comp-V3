Beginner V5 Program Features

All the features you would want in a beginner V5 program... and then some!

üåÄ Gyro Calibration

  • Function: calibrateGyro()
  • Description: Easily calibrate the gyro sensor for accurate readings.
  • Example: calibrateGyro();
  • Return Value: None.

üìÅ Config Manager

  • Methods:

    • Get Current Logging Setting:

      • Function: getLogToFile()
      • Description: Check if logging to a file is enabled. Returns true if enabled, false otherwise.

    • Set Logging Configuration:

      • Function: setLogToFile(bool logToFile)
      • Parameters: logToFile (type: bool) - Set it to true to enable logging and false to disable.

üîò Button Press Detection

  • Function: ctrl1BttnPressed()
  • Description: Detect pressed buttons on the primary controller and retrieve the button pressed along with the duration.
  • Example: 
    auto [button, duration] = ctrl1BttnPressed();
// OR if you only want the button!
std::string buttonPressed = std::get<0>(ctrl1BttnPressed());
  • Return Value: Pair of button pressed and duration.

üìä User Options Selection

  • Function: getUserOption()
  • Description: Prompt users for a setting choice and retrieve their selection.
  • Example Use: 
    int value = getUserOption("Choose a setting", {"Option 1", "Option 2"});
  • Return Value: User-selected option. (Ex: Option 1)
